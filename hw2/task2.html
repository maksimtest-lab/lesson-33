Задание 2

Реализуйте пример сложного вычисления, например, вычисление числа Фибоначчи с использованием WebAssembly.

Для компиляции в WebAssembly я использовал команду докера

docker run --rm -v $(pwd):/src -u $(id -u):$(id -g) emscripten/emsdk emcc fibonacci.c -o fibonacci.js -s EXPORTED_FUNCTIONS="['_fibonacci']" -s MODULARIZE -s ENVIRONMENT=web

В качестве примера использовал простое вычисление числа Фибоначчи на С - fibonacci.c

#include 

uint32_t fibonacci(uint32_t n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}
